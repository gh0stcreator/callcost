CallCost

Privacy Policy

Effective date: April 8, 2026

Overview

CallCost is a Zoom app that estimates the live cost of a meeting. This version is designed to work primarily inside the Zoom client and uses minimal data to render the in-meeting experience.

Information We Process

CallCost processes the following categories of information:

  • User-entered salary and workday values.
  • Basic Zoom meeting context that is available to the app within the Zoom client.
  • Basic local app state needed to preserve the user experience between app sessions on the same device.

How Data Is Used

  • To calculate and display an estimated meeting cost in real time.
  • To display the user's own contribution and timer state.
  • To restore locally saved input values for convenience on the same device and browser environment.

Storage and Retention

User-entered salary information is stored locally in the browser or Zoom webview on the user's own device. It is not intended to be stored on a remote server by this version of the app. Local data remains until the user clears browser or app storage, resets the app, or overwrites the saved values.

Sharing

CallCost does not sell personal information. This version is not designed to share salary data through Zoom APIs. Any display of meeting data is intended only for the in-meeting user experience inside Zoom.

Data Subject Rights

Depending on your location, you may have rights to request access, correction, deletion, restriction, objection, or portability for personal information processed by the app.

Because this version primarily stores salary input locally on the user's own device, users can usually exercise deletion directly by clearing local browser or app storage. If you need help exercising privacy rights or have questions about data handling, use the support page linked below.
